# 🌾 Agriculture Land Leasing Website

A full-stack web platform that connects farmers and 
landowners for agricultural land leasing.

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Database: MySQL

## ✨ Features
- 🏡 Land listing and browsing
- 📅 Online booking system
- 👤 User registration & login
- 🔍 Search and filter lands

## 🚀 How to Run
1. Clone the repository
2. Run `npm install`
3. Run `node server.js`
4. Open `index.html` in browser
